# 🚀 English

> **"최소한의 노력으로 최대의 결과를 얻는다."** > 복잡한 설정 없이 바로 학습하고, 다른 사람의 단어장을 1초 만에 가져오는 스마트 영어 학습 플랫폼입니다.

## ✨ 주요 기능
- **📂 세트 관리**: 나만의 단어장을 만들고 관리합니다.
- **🛒 세트 스토어**: 다른 사용자가 공유한 단어장을 클릭 한 번으로 내 서재에 복사합니다.
- **📖 3단계 학습 모드**:
  - **단어장**: 카드를 뒤집으며 단어를 암기합니다.
  - **스크램블**: 뒤섞인 단어를 조합해 문장을 완성합니다.
  - **스피킹**: 음성 인식을 통해 발음을 교정하고 말하기 연습을 합니다.
- **☁️ 실시간 동기화**: Supabase를 사용하여 어디서든 내 단어장을 열어볼 수 있습니다.

## 🛠 Tech Stack
- **Frontend**: React, Vite, React-Router-Dom
- **Backend/DB**: Supabase (PostgreSQL, Auth)
- **Deployment**: Vercel

## 🚀 빠른 시작
1. `npm install` (의존성 설치)
2. `npm run dev` (로컬 실행)
3. `npm run build` (배포 파일 생성)
